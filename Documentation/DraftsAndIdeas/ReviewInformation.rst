=======================
Show Review Information
=======================

Some ideas for showing review information on top of pages.

* document state: needs language improvements | outdated
* last checked on <date>, by <reviewer> for <TYPO3 version>
* if information on page


Suggestion 1: Add a Note
========================


Example for content, that is ok:

.. note::

   **Review information:**

   * Document state: correct
   * Last reviewed: May 15 2019 for TYPO3 version 9.5.1


Example for content, that is not ok:

.. warning::

   The information on this page is outdated!
